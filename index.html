☁ Setup Cloudinary (Untuk Upload Foto & Video Besar)

📋 Cara mendapatkan Cloud Name & Upload Preset:
1. Daftar gratis di cloudinary.com
2. Setelah login, lihat Cloud Name di dashboard (pojok kiri atas)
3. Buka Settings → Upload → Upload presets
4. Klik "Add upload preset", set Signing Mode = Unsigned, lalu simpan
5. Salin nama preset tersebut dan tempel di bawah

✨ Gratis 25GB storage — cukup untuk ratusan foto & video!

📌 Panduan Upload di Tab Galeri

• Setelah config tersimpan, buka tab 📸 Galeri
• Pilih foto / video sebesar apapun (tidak ada limit dari sisi kita)
• Cloudinary mendukung: JPG, PNG, GIF, WebP, MP4, MOV, AVI, dll
• File akan diupload langsung ke cloud dan URL-nya tersimpan
• Foto & video bisa dilihat semua orang yang membuka website 🎉

🔥 Setup Firebase Realtime Database

📋 Cara mendapatkan Firebase Config (ikuti urutan ini):
1. Buka console.firebase.google.com → klik Add project
2. Beri nama project (misal: xii-tjkt1-2026) → Continue → Create project
3. Di sidebar kiri klik Build → Realtime Database → Create database
4. Pilih region asia-southeast1 (Singapore) → Next
5. Pilih "Start in test mode" → Enable
6. Salin URL database (contoh: https://xii-tjkt1-default-rtdb.asia-southeast1.firebasedatabase.app)
7. Klik ikon ⚙ gear → Project settings
8. Scroll ke bawah ke "Your apps" → klik icon </> (Web)
9. Register app (isi nickname, tidak perlu hosting) → salin seluruh firebaseConfig
10. Isi form di bawah → Simpan → selesai! 🎉

✨ Gratis selamanya untuk proyek kecil — pesan semua orang tersimpan realtime!

ℹ️ Cara Baca firebaseConfig

const firebaseConfig = {
  apiKey: "AIzaSyXXXXXXXX", ← masuk ke API KEY
  authDomain: "xxx.firebaseapp.com", ← AUTH DOMAIN
  databaseURL: "https://xxx.firebasedatabase.app", ← DATABASE URL
  projectId: "xxx", ← PROJECT ID
  appId: "1:xxx:web:xxx" ← APP ID
};
